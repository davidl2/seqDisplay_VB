This is a simple exercise building a REST web-service client
using Visual Basic .NET.

The "service" it consumes (endpoint currently hardcoded) produces a plain-text document containing an integer.

Dependencies
------------

The .NET SDK, dowload from...
  https://dotnet.microsoft.com/en-us/download/dotnet/6.0 .

On RedHat Linux, it's also in package "dotnet-sdk-6.0".

Building
--------

Run the following command:

dotnet build

The compiled program will be in folder bin/Debug/net6.0/ .
